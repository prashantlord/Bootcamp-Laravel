## About Project

"This is a simple project created to learn how to develop a production-grade Laravel application."

### Packages Used
- **Laravel Breeze**
```bash
# Install Laravel Breeze
composer require laravel/breeze

# Install Breeze with API scaffolding
php artisan breeze:install api
```

### Current Progress
- Login/ Register system implemented as of 2025/12/09

### Dependencies 
```bash
PHP: 8.4.15
Composer: 2.8.12
Laravel: 5.23.1
Laravel: 12
```


### Running the Server
Run this command to download all the dependencies

```bash
$ composer install
```


Start the development server using Laravel's built-in server:

```bash
$ php artisan serve
```

Or use the Composer script:

```bash
$ composer run dev
```

Visit `http://127.0.0.1:8000` in your browser.
